◆ ncx_time_delta_init()

void ncx_time_delta_init ( void  )

Initializes the delta time variables.

Stores the current time in the "time_last" variable used for calculating delta-time. This is so there's not a massive spike of speed right as the application is booting up. Just make sure this is called before your main loop so there's as little time as possible between when it gets calculated next.
